FlyWire Connectome data (v783) \
This folder will contain the modularity ids of the **input connectome of both the oviIN instances (right and left)**, thresholded at 3 connections between a unique pair of neurons (undirected).

For each file name:
The **fw** in the file titles stands for **FlyWire**. \
The "thresh" designates that this version was thresholded. \ 
The oviIN notes the neurons (oviIN_r and oviIN_l) which the subconnectome revolves around. \
The number denotes the chi resolution used:
- 0 => 0.0
- 005 => 0.05
- 01 => 0.1
- 025 => 0.25
- 05 => 0.5
- 075 => 0.75
- 1 => 1.0
